# Supabase setup

## Застосування схеми

1. Відкрити Supabase Dashboard → SQL Editor.
2. Скопіювати та виконати файл:
   `supabase/migrations/20260628_create_roadmap_items.sql`.
3. Переконатися, що таблиця `public.roadmap_items` з’явилася в Table Editor.
4. Після першого відкриття сайту таблиця автоматично заповниться стартовими рядками.

Migration створює:

- таблицю для змінних даних карток;
- автоматичне оновлення `updated_at`;
- RLS-політики читання, створення й редагування;
- підключення таблиці до Supabase Realtime.

## Модель доступу

Поточна версія сайту не має авторизації. Тому publishable key отримує публічний
доступ до читання й редагування `roadmap_items` через RLS. Не додавайте до цієї
таблиці приватні дані. Коли з’явиться авторизація, політики потрібно обмежити
роллю `authenticated`.
